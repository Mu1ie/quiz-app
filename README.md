# 🧠 Interactive Quiz App

A simple interactive quiz app built as my first real GitHub project.

---

## 📖 Background

I was learning JavaScript and wanted to build something real —
not just exercises, but an actual deployable project.

I worked through this with **Claude Sonnet** by Anthropic as a mentor,
who guided me step by step: from project structure and writing the logic,
all the way to deploying it live on GitHub Pages.

---

## 🌐 Language Note

> This app is built in **Arabic** — questions, UI, and all content are in Arabic (RTL layout).
> If you don't read Arabic, the structure and code are still fully readable.

- Git basics — commit, push, deploy
